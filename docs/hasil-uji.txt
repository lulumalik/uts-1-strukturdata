========================================
  ANALISIS BIG O ΓÇö KLINIK E-HEALTH
========================================

Struktur Data:
  Array akses indeks     : O(1)
  Array/LL sequential    : O(n)
  Queue enqueue/dequeue  : O(1) / O(n)*
  Stack push/pop         : O(1)
  Tree cari poli         : O(n)
  Graph BFS / DFS        : O(V + E)
  *dequeue menggeser elemen (implementasi array)

Sorting:
  Bubble Sort            : O(n^2)
  Selection Sort         : O(n^2)
  Insertion Sort         : O(n^2) worst, O(n) best

Searching:
  Linear Search          : O(n)
  Binary Search          : O(log n) ΓÇö butuh data terurut

Manfaat bagi pengguna e-health:
  - Sorting memudahkan petugas melihat daftar pasien teratur
  - Binary Search mempercepat pencarian No RM pada data besar
  - BFS memberi jalur rujukan terpendek antar poli
